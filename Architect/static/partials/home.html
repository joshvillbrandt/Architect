rchitect

Architect is an avionics system design tool.

Browse the library

Parts

Architect is built around parts. Parts have channels which are assigned to external connectors.

Channels

Parts can also have internal child parts. You can define the relationship between parts using by mating channels between parts and creating harnesses to implement those channel allocations.

Connectors

Once you have defined some parts, you can get the data out of Architect using its API. This allows you to build additional applications on top of Architect.
